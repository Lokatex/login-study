# rest-api-fetch

Aby uruchomić projekt, w katalogu głównym wykonaj następujące komendy:

```
npm install
npm run dev
```

Gdy zobaczysz następujący komunikat to znaczy, że wszystko działa poprawnie i możesz zabrać się do pracy:

